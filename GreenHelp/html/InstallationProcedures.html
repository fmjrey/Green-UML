Installation for Green 3.x.x:

System requirements:
- Eclipse version 3.4
- GEF(Graphical Editing Framework) 3.4 (While GEF 3.5 is available for download, it is not the latest stable version, and it is known to be incompatible with Green) Get it here.

Installation through Eclipse update system (Preferred Method):
- Starting with the 3.1.0 release of the Green UML Eclipse plugin, installation and updates can be done through eclipse
- Simply open Eclipse, and navigate to Help -> Software Updates...
- Click on Add Site... on the right hand side
- Type in http://www.cse.buffalo.edu/faculty/alphonce/green/ and click Ok



- At this point, the Green Update Site should be listed as an available software location



- Click the check box next to the Update Site and click Install
- Please note that the relationships may have a different version number than the main version of Green. This is normal. The SVG save option will also most likely have a different version number.
- Follow the dialogues, and Green will be installed!
- Restart Eclipse and everything should be functioning properly


Installation through file system drop-in:
- After downloading the Green zip file, extract the contents to [Eclipse installation directory]\dropins\
- If warned that any existing files with the same name will be overwritten, select "Yes to All"
