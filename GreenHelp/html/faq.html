Frequently Asked Questions

 

When I try to open a new diagram, Eclipse tells me that "The chosen operation is not currently available." What do I do?
In order for Green to run properly, it is necessary that you have Java 5 installed and that Java 5 is used to load the Green plugin.

How do I save my diagram as a GIF/JPEG image file?
In the 'Save As...' menu, the dropdown for file type has four options: Green File, GIF, JPEG & PNG. Choosing one of the image options will export your file in that image format.
Please Note: In order for the Green 'Save As...' menu option to be available, the Green editor window needs to be open.
