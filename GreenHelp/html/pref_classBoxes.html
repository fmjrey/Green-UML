Preferences - Class Boxes

Fixed Height / Fixed Width
Defines the height and width of class boxes in digrams when using the 'Set to Fixed Size' context menu command.

Show fully-qualified names
Class names in class boxes will include their respective packages.

Show method parameter names
Method parameter names will be shown in class boxes.

Show tooltips while hovering over elements
Tooltips containing JavaDoc for specific elements will be displayed while hovering over Java elements in diagrams.
