General Preferences

Create all diagram files in project root
When generating new diagrams, the file will placed in the project root.

Generate stubs for public void main
Automatically check this option in New Class Wizard.

Generate stubs for constructors from superclass
Automatically check this option in New Class Wizard.

Generate stubs for inherited abstract methods
Automatically check this option in New Class Wizard.

Grid Size
Sets the size of the background grid, in pixels, to which elements of diagrams will snap.

Save compilation units automatically after code modification
By saving compilation units automatically after the code is modified, Green has a much quicker response time to changes. However, if the compilation unit is not saved automatically, Green will not update until it is saved.

Show subtype names on relationship arcs
Labels different relationships differently which are drawn with the same type of relationship arcs.

Use Manhattan routing
Automatically create fixed bendpoints at 90 degree angles. See example here.

Display Incremental Explorer Icons in Diagram Editor
Sets whether Class, Enum, and Interface boxes display incremental explorer icons. Using the icons, You can incrementall explore for either all relationship types, or only certain ones.

Relationship Line Width
Sets the width of the relationship lines drawn in diagrams.
