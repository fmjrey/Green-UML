Creating Classes and relationships with the Green editor

Classes can be created through Green by selecting the 'Class' label, and clicking inside an active diagram editor.

At this point, the Eclipse 'New Class Wizard' will initialize.

When classes exist in a diagram, relationships can be created using the appropriate relationship labels on the left panel.

The relationship will be generated by first clicking on the source class, then the target class. Any relationship-specific dialog prompts may appear for additional information. Afterwards, the appropriate code will be inserted into the class files.
